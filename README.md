# Ownit Rent A Car Website

This repository contains the files for the Ownit Rent A Car website.

## Purpose
This website is for a car rental business in Islamabad, Pakistan.  
It allows users to:
- View available cars
- Check rental prices
- Book cars via WhatsApp

## Pages
- Home
- Our Cars
- Booking
- Contact

## Deployment
This website is hosted for free on GitHub Pages.

**Live site:** https://hamdanrentacar.com

## Technologies Used
- HTML
- CSS
- JavaScript
- GitHub Pages (hosting)

## Features
- Car listings with images
- Pricing per day
- WhatsApp booking button
- Google Maps location
